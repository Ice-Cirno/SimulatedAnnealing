# SimulatedAnnealing @201705
模拟退火类解决TSP、0-1背包问题

平台：Visual Studio 2017, Windows subsystem for Linux.
编码：ANSI
makefile在Windows subsystem for Linux平台可用

进度：目前已完成TSP问题，0-1背包正在完成...

基类SA实现抽象算法
派生类TSP
main()会将最后退火的得到的最优解输出，如果#define DEBUG可在最优解更新时将其输出
可在main.cpp中更改所用的测试样例。

测试样例放在sample-input-files文件夹下。
